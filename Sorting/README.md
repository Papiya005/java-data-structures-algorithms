# Merge Sort in Java

## Overview

This project implements the Merge Sort algorithm in Java using the Divide and Conquer approach.

Merge Sort recursively divides the array into smaller subarrays, sorts them, and then merges them back together to produce a sorted array.

---

## Algorithm

1. Divide the array into two halves.
2. Recursively sort the left half.
3. Recursively sort the right half.
4. Merge the two sorted halves.

---

## Example

### Input

```text
[6, 3, 9, 8, 2, 5]
```

### Output

```text
[2, 3, 5, 6, 8, 9]
```

---

## Time Complexity

| Case         | Complexity |
| ------------ | ---------- |
| Best Case    | O(n log n) |
| Average Case | O(n log n) |
| Worst Case   | O(n log n) |

---

## Space Complexity

```text
O(n)
```

A temporary array is used during the merge process.
